# task-manager
Petit site Web permettant d'organiser des tâche en catégories et de supprimer celles qui ne sont plus pertinentes.

Le principal problème actuellement est que la sauvegarde marche tant que le serveur est ouvert.
Moi qui utilise live server, après une fermeture du serveur local, il n'y a pas de données de backup.
C'est quelque chose qu'il reste à travailler.
